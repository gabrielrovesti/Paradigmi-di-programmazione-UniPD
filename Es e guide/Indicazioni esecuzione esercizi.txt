Per seguire con profitto il corso è consigliato avere a disposizione un computer su cui installare:

Java JDK versione 17
Un Integrated Development Environment (IDE)
Un client per il sistema di versionamento Mercurial
Come distribuzione del Java Development Kit è preferita la Eclipse Adoptium Temurin 17. In alternativa è possibile usare la distribuzione Oracle o la Azul Zulu. In ogni caso, è caldamente consigliato l'uso di un package manager come per esempio chocolatey per Windows, homebrew per MacOs o quello di sistema (yum, apt, rpm) su Linux; in questo modo installare, aggiornare o rimuovere il JDK sarà molto più semplice.

Come IDE sono consigliati, in ordine di preferenza: Microsoft Visual Studio Code, Eclipse Java IDE e JetBrains Idea. La compilazione e l'esecuzione del codice presentato non sarà dipendente dall'IDE scelto, quindi la scelta è lasciata allo studente.

---Importante---
Aprire nella cartella src per beccarsi tutti i file di progetto
e quelli di test, poi eseguirli partendo da un file .java che dispone di un metodo main.
In questo modo gli esercizi andranno regolarmente. 

Eventualmente aprire solo dall'IDE/Vs Code la cartella main e si sarà
comunque in grado di eseguire i file singoli.
